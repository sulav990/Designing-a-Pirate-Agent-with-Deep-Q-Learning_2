# Designing-a-Pirate-Agent-with-Deep-Q-Learning_2
CS 370
For this project, I worked on developing an intelligent agent for a treasure hunt game where a pirate character navigates a maze to find a treasure. I was provided with the starter code that built the game environment, defined the maze layout, and set up the basic graphical interface. The code I created myself was the deep Q-learning algorithm. I wrote the logic that allowed the pirate to explore the environment, learn from a system of rewards and penalties, and ultimately discover the optimal path to the treasure on its own.

Looking back at this course, it is easy to see how this ties into the broader world of computer science. At its core, computer scientists are problem solvers. They build systems, automate tedious tasks, and create software that improves how we live and work. This matters because the solutions computer scientists build drive almost every modern industry, from healthcare and logistics to entertainment and communication. We take complex, real-world issues and translate them into a language that machines can understand and process.

When I approach a problem as a computer scientist, I start by breaking it down into smaller and more manageable pieces. Instead of getting overwhelmed by the massive end goal, I define the specific inputs I have and the exact outputs I need. Then, I use logic to bridge that gap step by step. It involves a lot of logical thinking, testing, and debugging. If an approach fails, I analyze what went wrong, refine my algorithm, and try again until the solution is efficient and reliable.

Finally, doing this work comes with serious ethical responsibilities. To the end user, my duty is to create software that is safe, fair, and respectful of their privacy. To the organization, my responsibility is to write code that is secure, clean, and maintainable.
